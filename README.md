# TEST_DTPS_G5
Repositorio de pruebas para el programa de gestion de vehiculos
## dependencias 
- selenium 4 
- python 3
- webdriver_manager
- google chrome instalado
## instalacion 
### selenium via pip 
<code>pip install selenium</code>
### webdriver via pip
<code>pip install webdriver-manager</code>
